# proyecto_universidad

Aplicar los conceptos fundamentales de HTML5, CSS3 y diseño responsive para desarrollar las interfaces de usuario de un sistema web de gestión de solicitudes de crédito, implementando buenas prácticas de maquetación, estructura semántica y diseño profesional.

Adrian Silva Henao

Descargar el archivo .rar llamado proyecto_creditos, descomprimir el archivo y moverlo en tu carpeta llamada vs_code, abrir Visual Studio Code y instalar la extension de Live Server, click derecho en el index.html y darle click en Ejecutar en Live server.
